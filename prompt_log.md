### P001 · 2026-07-13

**对应文件/功能**：`backend/services/ai_service.py` — 灵感碎片返回结构设计

**我的 Prompt**：
```
我在做一个写作辅助工具，有一个功能叫"灵感碎片骰子"。
用户输入一个主题后，AI 需要返回三类写作碎片：
1. sensory 感官细节（4条，眼耳鼻舌身意的具体印象）
2. scenes 微场景元素（3条，日常但有张力的小画面）
3. conflicts 微小冲突点（3条，心口不一/欲言又止类的小状况）

请帮我设计这个功能的 system prompt，要求：
- 明确 AI 的角色是"灵感骰子"，不是代写者
- 返回严格 JSON，键名就是 sensory/scenes/conflicts
- 每条都是一两句话的小点子，不要写大段
```

**AI 原始输出**：
```
system = """你是一个「灵感骰子」。围绕给定的主题，抛出三类写作碎片（都是一句话的小点子，作者自己挑选融入）：
1. sensory: 4条左右的感官细节（眼耳鼻舌身意的具体印象）
2. scenes: 3条左右的微场景（日常但有张力的瞬间画面）
3. conflicts: 3条左右的微小冲突/心口不一/欲言又止的小状况
严格返回 JSON，键名 sensory/scenes/conflicts。
不要写大段文字，每条一两句话。"""
```

**备注**：采纳了 90%，直接用于 `ai_service.py` 中 `inspiration_fragments()` 函数的 system prompt。

需要在写作页顶部放一个醒目的提示条，用暖色（琥珀/黄色系）。
帮我写一段 2-3 句的提示文案，要求：

### P003 · 2026-07-13

**对应文件/功能**：`frontend/app/write/page.js` — 「表达参考」模块的卡片设计

**我的 Prompt**：
```
我有一个 React 组件，展示「表达角度参考」。
每条参考包含三个字段：
- angle: "感官描写切入"（角度名）
- structure: "先写A → 再点出B → 补一个C动作"（句式结构）
- keywords: ["气味", "温度", "忽然", ...]（关键词参考）

我不要提供"一键复制"，因为我们的平台不让直接复制AI生成的成品句，
但这三条都是参考结构和关键词，可以提供复制。
帮我设计一个 Tailwind 的卡片样式，信息层级清晰，
包含角度名（带序号徽章）、思路行、关键词行，整体尺寸紧凑但不拥挤。
```

**AI 原始输出**：
```jsx
{aiPanel.result.map((ref, i) => (
  <div key={i} className="p-3 rounded-lg bg-white border border-ink-100 space-y-1.5">
    <div className="flex items-center gap-2">
      <span className="text-xs font-bold px-1.5 py-0.5 rounded bg-ink-100 text-ink-600">角度 {i + 1}</span>
      <span className="text-xs font-medium text-accent-600">{ref.angle}</span>
    </div>
    <p className="text-xs text-ink-500">思路：{ref.structure}</p>
    <p className="text-xs text-ink-400">关键词参考：{ref.keywords.join('、')}</p>
  </div>
))}
```

**备注**：采纳了 100%，直接用于写作页 AI 面板的表达参考展示。
